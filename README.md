# GEMCV – AI Resume Analyzer

GEMCV is a modern AI-powered Resume Analyzer platform that helps users evaluate and improve their resumes for better ATS compatibility and professional impact. The application provides intelligent resume analysis, resume scoring, improvement suggestions, and secure authentication with a clean and responsive user interface.

---

## 🚀 Features

- 🔐 Secure Firebase Authentication (Login & Signup)
- 📄 Resume Upload and PDF Support
- 🤖 AI-Powered Resume Analysis
- 📊 ATS Resume Scoring System
- 💡 Smart Resume Improvement Suggestions
- 🎨 Modern Responsive User Interface
- ⚡ Fast and Optimized Performance
- ☁️ Cloud Deployment Support
- 🔒 Secure Backend API Integration

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Firebase Authentication
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

### Deployment
- Frontend: Vercel / Render
- Backend: Render

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/gemcv.git
cd gemcv
```

---

### 2️⃣ Frontend Setup

```bash
cd client
npm install
npm run dev
```

---

### 3️⃣ Backend Setup

```bash
cd server
npm install
npm start
```

---

## 🔑 Environment Variables

### Backend `.env`

```env
MONGO_URI=your_mongodb_connection_url
JWT_SECRET=your_secret_key
```
